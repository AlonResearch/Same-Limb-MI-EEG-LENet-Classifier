# Machine learning for MI-EEG Classification
Using mostly LENet Light Wheight Convolutionnal neural networks by replacing the Fully connected layer before Classification to a Complete Convuted Block to classify of MI-EEG data

## Requirements
pytorch 1.12.1
scikit-learn 1.5.1
scipy 1.13.1

## File list

	SNN_for_EEG_based_Motor_Imagery_Classification.ipynb: All the scripts in one file


### DataSets
  [BCICIV_2a_gdf](https://www.bbci.de/competition/iv/download/index.html): The preprocessed dataset for testing the framework
